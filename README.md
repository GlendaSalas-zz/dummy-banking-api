# Dummy banking app
A simple dummy web-based banking application
# Add enviroment variables
- Add a .env file with the following variables
    - VERSION = v1
    - PORT = 8000
    - PROJECT_NAME = Dummy banking REST API
    - MONGO_DEV = **********
    - JWT_SECRET = ********
    - TIME_EXPIRES = 8h
# Run local
- `npm install`
- `npm run dev`
- `localhost:PORT`
# Run test
- `npm run test`
# Project url
https://dummy-banking-api.herokuapp.com/v1

# Stack:
- MongoDB
- Node.js
- Express.js
